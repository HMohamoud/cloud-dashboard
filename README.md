# ☁️ Cloud Resource Monitoring Dashboard

A **real-time cloud monitoring dashboard** using AWS services (EC2, S3, CloudWatch).  
Visualises CPU usage and storage metrics for your cloud resources in a **modern, responsive web interface**.

![Dashboard Screenshot](static/img/screenshot.png)

## Features

- Live **EC2 CPU monitoring**
- Live **S3 storage visualisation**
- Flask web app with modern **Bootstrap-based dashboard**
- Clean, responsive design
